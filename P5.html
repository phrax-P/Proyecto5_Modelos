P5 - Cadenas de Markov¶

Como un caso especial de los procesos aleatorios donde existen "estados" y "transiciones" entre estos estados, las cadenas de Markov modelan las probabilidades de transicin y el tiempo de permanencia entre cada estado. En particular, la teora de colas describe la evolucin del nmero de elementos en la fila, dado un flujo de entrada y un flujo de salida.

1. - Procesos de nacimiento y muerte¶

Los procesos de nacimiento y muerte son un caso especial de las cadenas de Markov donde las transiciones solamente pueden ocurrir de un estado $i$ a un estado $i + 1$ ("nacimiento" o llegada) o $i - 1$ ("muerte" o salida).

  • Las llegadas estn modeladas con un parmetro $\lambda$ de un proceso de Poisson, tambin llamado "intensidad".
  • Las salidas, o tiempo de servicio, estn modeladas exponencialmente con un parmetro $\nu$.

Dos componentes de este sistema son aleatorios:

  • El tiempo que tarda "la mquina" (el sistema) en el estado $i$, que est modelado exponencialmente con un parmetro $\Omega_i$.
  • La transicin que har de ese estado, que puede ser "hacia arriba" ($i+1$) con probabilidad $p_i$ o "hacia abajo" ($i-1$) con probabilidad $q_i$.

Generalmente es necesario encontrar la expresin para $\Omega_i$, $p_i$ y $q_i$ en trminos de $\lambda$ y $\nu$ para conocer mejor la dinmica del sistema.

2.1. - Problema de la gasolinera¶

Supngase que hay una gasolinera con tres entradas, los autos que entran arriban en forman de una corriente de Poisson con parmetro $\lambda$. La entrada norte tiene tasa de llegada de dos autos por minuto, la entrada este tiene una intensidad de dos autos por minuto y la entrada sur tiene un parmetro de un auto por minuto. El tiempo de servicio de cada dispensador de gasolina est exponencialmente distribuido con parmetro $\nu$ autos por minuto. El estado $X_t$ en el tiempo $t$ es la longitud de la cola. Inicialmente, en la gasolinera se enuentran tres dispensadores de gasolina funcionando pero, si la fila sobrepasa los cinco automviles, abren otros dos dispensadores ms.

Suponga que el parmetro $\nu$ es la mediana entre los ltimos dgitos del nmero de carn universitario de quienes realizan este proyecto. Ejemplo: A13843, B83648 y C23177 tienen una mediana de 7.

import numpy as np
import matplotlib.pyplot as plt



def random_walk(muestras):

    # Parmetros
    lmbda = 5
    nu = 2 # X

    # Arrays con historial de estados y tiempo en cada uno
    estados = np.zeros(muestras)
    tiempos = np.zeros(muestras)
    # Iniciamos con 0 carros en la fila
    prev_state = 0
    Omegas = [[], [], [], [], []]

    # Ciclo del random walk
    for i in range(muestras):
        # Caso i=0
        if prev_state == 0:
            curr_state = 1 # nicamente puede crecer la cola
            Omega = lmbda
            tiempos[i] = np.random.exponential(1/Omega) # alv tremenda funcin :V np.random.exponential(parmetro) la verdad si :o
            Omegas[0].append(tiempos[i])
        # Caso 1
        elif (prev_state == 1):
            curr_state = np.random.choice((prev_state-1, prev_state + 1), p = (nu/(lmbda + nu), lmbda/(lmbda + nu))) # np.random.choice(evento, prob)
            Omega = lmbda + nu
            tiempos[i] = np.random.exponential(1/Omega)
            Omegas[1].append(tiempos[i])
        # Caso 2
        elif (prev_state == 2):
            curr_state = np.random.choice((prev_state-1, prev_state + 1), p = (2*nu/(lmbda + 2*nu), lmbda/(lmbda + 2*nu))) # np.random.choice(evento, prob)
            Omega = lmbda + 2*nu
            tiempos[i] = np.random.exponential(1/Omega)
            Omegas[2].append(tiempos[i])
        # Caso 3 <= i <=5
        elif (3 <= prev_state <= 5):
            curr_state = np.random.choice((prev_state-1, prev_state + 1), p = (3*nu/(lmbda + 3*nu), lmbda/(lmbda + 3*nu))) # np.random.choice(evento, prob)
            Omega = lmbda + 3*nu
            tiempos[i] = np.random.exponential(1/Omega)
            Omegas[3].append(tiempos[i])
        # Caso i>5 
        else:
            curr_state = np.random.choice((prev_state-1, prev_state + 1), p = (5*nu/(lmbda + 5*nu), lmbda/(lmbda + 5*nu)))
            Omega = lmbda + 5*nu
            tiempos[i] = np.random.exponential(1/Omega)
            Omegas[4].append(tiempos[i])
        
        estados[i] = prev_state
        prev_state = curr_state   
    Omega_inv = [np.mean(i) for i in Omegas]
    return estados, tiempos, Omega_inv
